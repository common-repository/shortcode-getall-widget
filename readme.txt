Add [Shortcodewidget widget="YOUR-WIDGET-NAME"] shortcode which enables you to output widgets anywhere you like in your page or post. 

== Description ==
Insert a entire widget area (sidebar) into your page without any line of code.But the any shortcode requires the widget ID, don't worry about to guess, the plugin get the code for you.

Here You can get code with using [Shortcodewidget widget="YOUR-WIDGET-NAME"] or select from editor.

== Installation ==

1. Download the plugin.
2. Upload the whole plugin directory to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Now, go to the page/post menu find Shortcode widget(button) and click or past [Shortcodewidget widget="YOUR-WIDGET-NAME"]
4. Enjoy!
